# Car Rental System

project:
  name: Car Rental System
  language: Java
  type: Console Application

description: >
  A simple Java based Car Rental System that allows users to view cars, rent a car, and return rented cars. The project demonstrates core
  Object-Oriented Programming concepts in a real-world scenario.

features:
  - Add and manage cars
  - View available vehicles
  - Rent a car
  - Return rented car
  - Menu driven console interface

tech_stack:
  - Java
  - Maven

project_structure:
  - Car.java : Represents car details
  - RentalService.java : Handles rental logic
  - Main.java : Application entry point

how_to_run:
  - Clone repository
  - Open project in IDE (Eclipse / IntelliJ / VS Code)
  - Run Main.java

learning_outcomes:
  - Java OOP concepts
  - Class interaction
  - Console input handling
  - Basic system design

future_improvements:
  - Database integration
  - User login system
  - GUI/Web interface
  - Billing system
